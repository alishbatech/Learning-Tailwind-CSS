# Learning Tailwind CSS 

This repository is dedicated to my daily practice, core concepts, and learning journey of **Tailwind CSS**.

## Project Structure
- `src/` - Contains the raw source code and components.
- `dist/` - Contains the compiled production-ready output files.

## Tools & Tech Stack
- HTML5 & Custom Layouts
- Tailwind CSS (Utility-first approach)
- Node.js & npm (for package management)

## Features Practiced
- Responsive web layouts using breakpoints (`sm:`, `md:`, `lg:`)
- Flexbox and CSS Grid alignments
- Custom themes, colors, and typography tracking
